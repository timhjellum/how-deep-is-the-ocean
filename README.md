# How Deep is the Ocean?

## Running Locally

### Prerequisites

- [Node.js](https://nodejs.org/) (v14+)

### Installation

```bash
git clone https://github.com/timhjellum/how-deep-is-the-ocean
cd how-deep-is-the-ocean
npm install
npm run watch
```

### Building & Deploying

```bash
npm run build       # Production build to dist/
npm run deploy      # Build + deploy to GitHub Pages
```
